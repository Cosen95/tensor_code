### 神经网络

#### 什么是神经网络？

- 人工神经网络是一种运算模型（就是输入输出的映射），由大量的节点（或称神经元）之间相互联接构成
- 每个神经元里存储着若干`权重`（weight）、`偏置`（bias）、和一个`激活函数`（activation）
- 输入乘上权重加上偏置，经过激活函数得到输出
- 激活函数用于添加一些非线性变换
